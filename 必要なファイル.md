# 必要なファイルリスト

以下のファイルが必要です。元のサイト https://letsmedia.vercel.app からダウンロードしてください。

## ダウンロード方法

### 方法1: ブラウザの開発者ツールから

1. Chrome で https://letsmedia.vercel.app を開く
2. F12 で開発者ツールを開く
3. Networkタブを選択
4. ページをリロード (Ctrl+R)
5. 以下のファイルを見つけて右クリック → 「Open in new tab」→ 右クリック → 「名前を付けて保存」

### 必要なファイル

#### 1. logoheda.jpg
- 用途: ヘッダーのロゴ画像
- URL: https://letsmedia.vercel.app/logoheda.jpg
- 配置場所: プロジェクトルート

#### 2. letsmedia読み込み.mp4
- 用途: ローディング画面の動画
- URL: https://letsmedia.vercel.app/letsmedia読み込み.mp4
- 配置場所: プロジェクトルート

#### 3. fabicon.png
- 用途: ファビコン（ブラウザタブのアイコン）
- URL: https://letsmedia.vercel.app/fabicon.png
- 配置場所: プロジェクトルート

#### 4. oGP.png
- 用途: OGP画像（SNSシェア用のサムネイル）
- URL: https://letsmedia.vercel.app/oGP.png
- 配置場所: プロジェクトルート

#### 5. Buildingsandundertherailwaytracksfree_ver.otf
- 用途: カスタムフォント
- URL: https://letsmedia.vercel.app/Buildingsandundertherailwaytracksfree_ver.otf
- 配置場所: プロジェクトルート

#### 6. company.html (オプション)
- 用途: 会社概要ページ
- URL: https://letsmedia.vercel.app/company.html
- 配置場所: プロジェクトルート

## 簡単ダウンロード方法

ターミナルやコマンドプロンプトで以下のコマンドを実行すると一括ダウンロードできます:

```bash
# プロジェクトフォルダに移動
cd /path/to/lets-media

# 各ファイルをダウンロード (curlまたはwget使用)
curl -O https://letsmedia.vercel.app/logoheda.jpg
curl -O https://letsmedia.vercel.app/letsmedia読み込み.mp4
curl -O https://letsmedia.vercel.app/fabicon.png
curl -O https://letsmedia.vercel.app/oGP.png
curl -O https://letsmedia.vercel.app/Buildingsandundertherailwaytracksfree_ver.otf
curl -O https://letsmedia.vercel.app/company.html
```

または wget を使う場合:

```bash
wget https://letsmedia.vercel.app/logoheda.jpg
wget https://letsmedia.vercel.app/letsmedia読み込み.mp4
wget https://letsmedia.vercel.app/fabicon.png
wget https://letsmedia.vercel.app/oGP.png
wget https://letsmedia.vercel.app/Buildingsandundertherailwaytracksfree_ver.otf
wget https://letsmedia.vercel.app/company.html
```

## ファイルダウンロード後の確認

全てのファイルをダウンロードしたら、プロジェクトフォルダの構成は以下のようになります:

```
lets-media/
├── index.html
├── style.css
├── main.js
├── company.html
├── logoheda.jpg
├── letsmedia読み込み.mp4
├── fabicon.png
├── oGP.png
├── Buildingsandundertherailwaytracksfree_ver.otf
└── README.md
```

これで完璧です！
